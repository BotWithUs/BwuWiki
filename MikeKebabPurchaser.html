MikeKebabPurchaser

   Version: v1.0.6   Author: Mike

Cost​

Cost

  • FREE / month (not including client access)

Features​

Features

  • A simple script with no account requirements to utilize.
  • Purchases Kebabs at Karim. Start the script next to Karim with 20k+ coins and you should be good to go for a while.
  • Open-source, not built with best programming practices in mind, but it is a simple functional script which should be easy to follow :) https://github.com/devMikeyB/MikeKebabBuyer
